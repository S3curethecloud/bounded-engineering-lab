# Lab Registry

This registry tracks bounded proofs, prototypes, deployment patterns, and reusable accelerators in this repository.

## Status Model

| Status | Meaning |
| --- | --- |
| Draft | Concept exists and is documented. |
| In Progress | Implementation is underway. |
| Validated | Required validation and evidence exist. |
| Accelerator Candidate | Lab is ready to be packaged for reuse. |
| Deprecated | Lab should no longer be used. |

## Labs

| Lab | Category | Status | Validation | Evidence | Accelerator Path |
| --- | --- | --- | --- | --- | --- |

| Bounded AI PR Reviewer | AI Workflow | Draft | Structure, boundary, markdown fence, AI boundary | Sample AI review evidence | accelerators/ai-workflow-prompts/bounded-pr-reviewer/ |

## Registry Rules

Every lab added to this repository should be registered here with:

- Lab name
- Category
- Status
- Validation method
- Evidence artifact
- Accelerator path, if applicable

## Promotion Path

Draft -> In Progress -> Validated -> Accelerator Candidate -> Reference Pattern
