# GitHub Actions Security Baseline Accelerator

## Status

Draft accelerator candidate

## Purpose

Reusable GitHub Actions baseline for early repository hardening.

## Includes

- CI structure validation
- Boundary validation
- Lightweight secret-pattern check
- Evidence summary generation

## Intended Use

Use this accelerator to bootstrap a repository with basic DevSecOps guardrails before adding language-specific scanning, SBOM generation, artifact signing, or deployment workflows.

## Boundary

This accelerator is not a production security program. It provides baseline workflow structure and reviewable evidence only.
